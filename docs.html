使い方ガイド

FCTZS TRPG部の特設サイトの使い方です。募集・キャラ・卓・予定調整などを Web で一元管理できます。

このサイトでできること: Discord ログイン、プレイヤー名簿との連携、キャラクター／シナリオ／卓の管理、予定入力と照合、募集、コメント・なりきりチャット。

1. はじめに

サイト URL: https://fctzs.daruji.workers.dev/

公開URL: 本番フロントは上記（Cloudflare Workers 静的配信）です。ブックマークや案内はこの URL を使ってください。

本システムは FCTZS ボドゲ部 Discord サーバーと連携しています。

  • 通知: 募集の作成・満員時、セッション当日のリマインドなどが Discord に届きます。
  • 気になる通知: シナリオ詳細で「気になる」を初めて ON にしたとき、そのシナリオを「GM可能」に登録しているプレイヤーへ Bot の DM が届きます。
  • ボタン操作: 募集案内の「参加希望」ボタンから、Web を開かずに応募できます。

2. はじめてのログインとプレイヤー連携

  1. 画面右上（またはホーム）の Discord Login を押します。
  2. Discord で認証するとサイトに戻ります。
  3. 部のプレイヤー名簿（players）にまだ紐づいていない場合、ホーム上部に 「プレイヤー連携が必要です」 バナーが出ます。

連携のしかた

  • 名簿に自分の Discord ID が既に登録されている場合は、自動連携されることがあります。
  • 自動連携されない場合は、バナーの一覧から自分の名前を選び「この名前で連携する」を押します。
  • Discord ID はバナーからコピーできます。取得できないときは一度 Logout → Discord Login し直してください。
  • 名簿に自分の名前が無い場合は、管理者に追加を依頼してください。
連携が完了すると個人ダッシュボード（次回予定・自分の募集・予定カレンダー）が表示されます。

3. ホーム（ダッシュボード）

  • 未ログイン: 部の直近予定・進行中セッションの概要と、Discord ログインへの誘導。
  • ログイン済み・未連携: 上記バナーで名簿と連携。
  • ログイン済み・連携済: 左に「自分の次回予定」「自分の募集」、右に予定カレンダー。下段に部全体の直近予定と進行中セッション。
予定カレンダーで赤く表示された日は予定未入力です。日付を選ぶと昼／夜をまとめて入力できます。

4. キャラクター

自分（または代理で他プレイヤー）のキャラクターを登録・管理します。

新規登録

  1. メニュー「Characters」→「新規登録」。
  2. 「プレイヤー」欄で所有者を選びます（代理作成では別メンバーを選べます。編集権限は作成したログインユーザーに紐づきます）。
  3. （任意）外部データをインポートできます。
    • クトゥルフ神話TRPG／新クトゥルフ神話TRPG: いあきゃら出力
    • エモクロアTRPG／ガイアケアTRPG: ココフォリア形式チャパレ
    • テキストの貼り付けに加え、.txt ファイルのアップロードからも反映できます
  4. システムを選び、プロフィール・能力値・技能・メモを入力して保存します。

編集

  • 詳細の編集から画像や基本情報を更新できます。
  • 能力値・技能は上書き保存でき、成長後もここから最新値に更新できます。
  • ココフォリア出力でチャットパレットをコピーできます。
画像: Web から直接アップロードできます（Cloudflare R2）。

5. シナリオ

部内で遊ぶ（または遊んだ）シナリオ情報を蓄積します。

登録と検索

  1. 「Scenarios」→「新規登録」で基本情報・詳細・傾向などを登録します。
  2. 一覧ではキーワード（タイトル・作者）、システム、人数、時間、ロスト率、傾向などで絞り込めます。
  3. 詳細からカバー画像や説明を編集できます。

気になる / GM可能

  • 気になる: ログイン後、シナリオ詳細で ON/OFF。興味人数も表示されます。OFF→ON（新規登録）時だけ、GM可能登録者へ Discord DM が送られます。
  • GM可能: 「このシナリオをGMできます」で自分のリストに追加。プレイヤー詳細の「GM可能シナリオ」からもまとめて登録できます。
お願い: ネタバレを含むメモなどは、未通過のプレイヤーが見てしまわないよう記載しないでください。

6. セッション（卓）

遊ぶシナリオと参加メンバーを紐付けた「卓」と、開催履歴を管理します。

新しい卓

  1. 「Sessions」→「新規卓作成」。
  2. シナリオ、GM、参加 PL を選んで保存します。

一覧ではキーワード（タイトル・シナリオ・GM・PL）、ステータス、システムで絞り込めます。

卓情報・セッション履歴

  • 卓詳細から参加キャラクター、カバー画像、配信／アーカイブ URL を登録できます。
  • セッション履歴に開催予定を追加できます。完了後はステータスが更新されます。
  • 観戦希望はセッション詳細からログインして登録できます（当日リマインドに反映されます）。

7. スケジュール

  • メニュー「Schedule」で直近予定や空き状況の照合ができます。
  • ホームの予定カレンダー、またはプレイヤー詳細からも予定を入力できます。
自動通知: セッション開始日の昼頃に、Discord へリマインドが送られます（GM・参加 PL・観戦希望者へのメンション）。

8. 募集

新規募集（GM向け）

  1. 「Recruit」→「新規募集」でシナリオ・役割・目標人数・メモを設定して公開します。
  2. 公開すると Discord の募集チャンネルに案内が投稿されます。

参加とキャンセル（PL向け）

  • Discord: 募集メッセージの「参加希望」ボタン。
  • Web: 募集詳細でログイン中の自分として応募／取消。

目標人数に達すると募集は「満員」になり、募集主へ Discord 通知が送られます。

9. プレイヤー詳細

  • プロフィール編集、プレイスタイルレーダー、推しキャラ／最強シナリオ。
  • PL通過済（部内卓＋部活外の個人履歴）／GM経験済シナリオ。
  • GM可能シナリオの登録（「気になる」通知の宛先）。
  • 所持キャラクター一覧とスケジュール。
部活外シナリオは Discord ログインなしで「部活外を追加」モーダルから登録でき、ほかのプレイヤーの履歴も追加・削除できます。シナリオ一覧には表示されません。

10. コメント・なりきりチャット

  • コメント: 各詳細画面や HOME の新着コメントで感想・やり取りを残せます。
  • なりきりチャット: 自分のキャラクター名義で掲示板に投稿できます（他人のキャラでは投稿不可）。

11. よくあること

  • ログインしたが個人画面が出ない: 名簿未連携です。ホーム上部のバナーから自分の名前で連携してください。
  • 名簿に自分の名前が無い: 管理者にプレイヤー追加を依頼してください。
  • Discord ID がコピーできない／取れない: Logout → Discord Login をやり直してください。
  • 予定カレンダーが赤い: その日の予定が未入力です。日付を選んで昼／夜を入力してください。
  • 気になるを押しても DM が来ない: すでに ON だった場合や、そのシナリオに GM可能登録者がいない場合は通知されません（新規 ON 時のみ）。
