FCTZS TRPG部 特設サイト - システム説明書

本システムは、FCTZS TRPG部の思い出を蓄積するための特設サイトです。募集の立ち上げから、キャラクターの管理、セッションのスケジュール調整までをWeb上で一元管理できます。

開発者向けのスキーマ・API説明は docs/DB-overview.md および docs/api-contract.md を参照してください。

1. はじめに

1-1. サイトへのアクセス

URL（https://fctzs.daruji.workers.dev/）にアクセスして利用します。

※公開URL: 本番フロントは https://fctzs.daruji.workers.dev/（Cloudflare Workers 静的配信）です。ブックマークや案内はこの URL を使ってください。

1-2. Discordとの連携について

本システムはFCTZSボドゲ部Discordサーバーと連携しています。

  • ログイン: Discord OAuth でサイトにログインします。名簿（players）への紐づけは次項のとおりです。
  • 通知機能: 募集の作成・満員時、セッション当日のリマインドなどが Discord に届きます。
  • 気になる通知: シナリオ詳細で「気になる」を初めてONにしたとき、そのシナリオを「GM可能」に登録しているプレイヤーへ Bot の DM が届きます（テスト設定時は実DMせずテスト用Webhookへプレビュー）。
  • ボタン操作: 募集案内の「参加希望」ボタンから、Webを開かずに応募できます。

2. ホーム（ダッシュボード）

ログイン状態によって表示が切り替わります。

  • 未ログイン: 部の直近予定・進行中セッションの概要と、Discordログインへの誘導。
  • ログイン済み・未連携: 上部に「プレイヤー連携が必要です」バナーが出ます。名簿から自分の名前を選び「この名前で連携する」を押してください。名簿に同じ Discord ID がある場合は自動連携されます。Discord ID のコピーもバナーから行えます（取得できないときは一度ログアウトして再ログイン）。
  • ログイン済み・連携済: 左に「自分の次回予定」「自分の募集」、右に予定カレンダー。下段に部全体の直近予定と進行中セッション（観戦・状況把握用）。
予定カレンダーで赤く表示された日は予定未入力です。日付を選ぶと一括入力できます。名簿に自分の名前が無い場合は管理者に追加を依頼してください。

3. キャラクター管理

自身（または代理で他プレイヤー）のキャラクター情報を登録・管理する機能です。

3-1. キャラクターの新規登録

  1. メニューから「キャラクター一覧」を開き、「新規登録」ボタンを押します。
  2. 「プレイヤー」欄で所有者を選びます。代理作成では別メンバーを選べます（編集権限は作成したログインユーザーに紐づきます）。
  3. （任意）「クトゥルフ神話TRPG」「新クトゥルフ神話TRPG」はいあきゃら出力、「エモクロアTRPG」「ガイアケアTRPG」はココフォリア形式チャパレをインポートできます。
  4. 対象システムを選択し、プロフィール、能力値、技能、メモを入力して保存します。

3-2. プロフィール・能力値・技能の編集

  1. キャラクター詳細の編集から、画像や基本情報を更新できます。
  2. 能力値・技能は上書き保存でき、成長後もここから最新値に更新できます。
  3. ココフォリア出力でチャットパレットをコピーできます。
【画像アップロード】 Cloudflare R2 連携により、Webから直接画像をアップロードできます。

4. シナリオ管理

部内で遊ぶ（または遊んだ）シナリオの情報を蓄積する機能です。

4-1. シナリオの登録と検索

  1. メニューの「シナリオ一覧」から「新規登録」で基本情報・詳細・傾向などを登録します。
  2. 一覧ではキーワード（タイトル・作者）、システム、人数、時間、ロスト率、傾向などで絞り込めます。
  3. 詳細画面からカバー画像や説明を編集できます。

4-2. 気になる / GM可能

  • 気になる: ログイン後、シナリオ詳細のボタンでON/OFFできます。興味人数も表示されます。OFF→ON（新規登録）時だけ、GM可能登録者へ Discord DM が送られます。
  • GM可能: 「このシナリオをGMできます」で自分のGM可能リストに追加できます。プレイヤー詳細の「GM可能シナリオ」からもまとめて登録できます。
※お願い： ネタバレを含むメモなどは、未通過のプレイヤーが見てしまわないよう記載しないでください。

5. セッション（卓）の管理

遊ぶシナリオと参加メンバーを紐付けた「卓」を管理し、実際のプレイ履歴を記録します。

5-1. 新しい卓の立て方

  1. メニューの「セッション（卓）一覧」から「新規卓作成」を押します。
  2. 遊ぶ「シナリオ」、担当「GM」、参加「PL」を選択して保存します。

一覧ではキーワード（タイトル・シナリオ・GM・PL）、ステータス、システムで絞り込めます。

5-2. 卓情報・セッション履歴

  1. 卓詳細から参加キャラクターやカバー画像、配信/アーカイブURLを登録できます。
  2. セッション履歴に開催予定を追加できます。完了後はステータスが更新されます。
  3. 観戦希望はセッション詳細からログインして登録できます（当日リマインドに反映されます）。

6. スケジュール機能

登録されたセッションの予定をカレンダー形式で確認し、空き日程を入力できます。

  • メニューの「スケジュール」で直近予定や空き状況の照合ができます。
  • ホームの予定カレンダー、またはプレイヤー詳細からも予定を入力できます。
【自動通知】 セッション開始日の昼頃に、Discord へリマインドが送られます（GM・参加PL・観戦希望者へのメンション）。

7. 募集機能

新しく卓を立てる際、参加者を募るための機能です。

7-1. 新規募集（GM向け）

  1. 「募集一覧」から「新規募集」を開き、シナリオ・役割・目標人数・メモを設定して公開します。
  2. 公開すると Discord の募集チャンネルに案内が投稿されます。

7-2. 参加とキャンセル（PL向け）

  • Discordから: 募集メッセージの「参加希望」ボタン。
  • Webから: 募集詳細でログイン中の自分として応募／取消。

7-3. 満員時

目標人数に達すると募集は「満員」になり、募集主へ Discord 通知が送られます。

8. プレイヤー詳細機能

各プレイヤーの活動状況を可視化・確認できる機能です。

  • プロフィール編集、プレイスタイルレーダー、推しキャラ／最強シナリオ。
  • PL通過済（部内卓＋部活外の個人履歴） / GM経験済シナリオ。
  • GM可能シナリオの登録（「気になる」通知の宛先）。
  • 所持キャラクター一覧とスケジュール。
部活外シナリオはDiscordログインなしで「部活外を追加」モーダルから登録でき、ほかのプレイヤーの履歴も追加・削除できます。シナリオ一覧には表示されません。

9. コメント・なりきりチャット

  • コメント: 各詳細画面や HOME の新着コメントで感想・やり取りを残せます。
  • なりきりチャット: 自分のキャラクター名義で掲示板に投稿できます（他人のキャラでは投稿不可）。
