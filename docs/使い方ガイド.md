**1. はじめに**

募集・キャラ・卓・予定調整などを Web で一元管理できます。

- 公開URL: `https://fctzs.daruji.workers.dev/`（ブックマークや案内はこの URL）
- Discord と連携しています
  - 募集の作成・満員時、セッション当日のリマインドなどが Discord に届きます
  - シナリオで「気になる」を初めて ON にしたとき、GM可能登録者へ Bot の DM が届きます
  - 募集案内の「参加希望」ボタンから Web を開かずに応募できます

---

**2. はじめてのログインとプレイヤー連携**

1. 画面右上（またはホーム）の **Discord Login** を押す
2. Discord で認証するとサイトに戻る
3. 部のプレイヤー名簿にまだ紐づいていない場合、ホーム上部に **「プレイヤー連携が必要です」** バナーが出る

**連携のしかた**
- 名簿に自分の Discord ID が既にある場合は、自動連携されることがある
- されない場合は、バナーの一覧から **自分の名前を選び**「この名前で連携する」を押す
- Discord ID はバナーからコピーできる（取れないときは Logout → Discord Login）
- 名簿に自分の名前が無い場合は、**管理者に追加を依頼**

連携が完了すると個人ダッシュボード（次回予定・自分の募集・予定カレンダー）が出ます。

---

**3. ホーム（ダッシュボード）**

- **未ログイン:** 部の直近予定・進行中セッションの概要とログイン誘導
- **ログイン済み・未連携:** 上記バナーで名簿と連携
- **ログイン済み・連携済:** 左に次回予定・自分の募集、右に予定カレンダー。下段に部全体の予定と進行中セッション

※ 予定カレンダーで赤い日は予定未入力。日付を選ぶと昼／夜をまとめて入力できます。

---

**4. キャラクター**

自分（または代理で他プレイヤー）のキャラを登録・管理します。

**新規登録**
1. メニュー「Characters」→「新規登録」
2. 「プレイヤー」欄で所有者を選ぶ（代理作成可。編集権限は作成したログインユーザーに紐づく）
3. （任意）外部データをインポート
   - クトゥルフ神話TRPG／新クトゥルフ神話TRPG: いあきゃら出力
   - エモクロアTRPG／ガイアケアTRPG: ココフォリア形式チャパレ
   - テキスト貼り付けに加え、**.txt ファイルのアップロード**からも反映可
4. システムを選び、プロフィール・能力値・技能・メモを入力して保存

**編集**
- 詳細から画像や基本情報を更新
- 能力値・技能は上書き保存（成長後もここから更新）
- ココフォリア出力でチャットパレットをコピー可
- 画像は Web から直接アップロード可

---

**5. シナリオ**

部内で遊ぶ（または遊んだ）シナリオ情報を蓄積します。

**登録と検索**
1. 「Scenarios」→「新規登録」で基本情報・詳細・傾向などを登録
2. 一覧ではキーワード（タイトル・作者）、システム、人数、時間、ロスト率、傾向などで絞り込み
3. 詳細からカバー画像や説明を編集

**気になる / GM可能**
- **気になる:** ログイン後、シナリオ詳細で ON/OFF。興味人数も表示。OFF→ON（新規）時だけ GM可能登録者へ Discord DM
- **GM可能:** 「このシナリオをGMできます」で自分のリストに追加。プレイヤー詳細からもまとめて登録可

※ ネタバレを含むメモなどは、未通過のプレイヤーが見てしまわないよう記載しないでください。

---

**6. セッション（卓）**

遊ぶシナリオと参加メンバーを紐付けた「卓」と開催履歴を管理します。

**新しい卓**
1. 「Sessions」→「新規卓作成」
2. シナリオ、GM、参加 PL を選んで保存

一覧ではキーワード（タイトル・シナリオ・GM・PL）、ステータス、システムで絞り込めます。

**卓情報・セッション履歴**
- 卓詳細から参加キャラ、カバー画像、配信／アーカイブ URL を登録
- セッション履歴に開催予定を追加。完了後はステータス更新
- 観戦希望はセッション詳細からログインして登録（当日リマインドに反映）

---

**7. スケジュール**

- メニュー「Schedule」で直近予定や空き状況の照合
- ホームの予定カレンダー、またはプレイヤー詳細からも予定入力

※ セッション開始日の昼頃に Discord へリマインド（GM・参加 PL・観戦希望者へのメンション）。

---

**8. 募集**

**新規募集（GM向け）**
1. 「Recruit」→「新規募集」でシナリオ・役割・目標人数・メモを設定して公開
2. Discord の募集チャンネルに案内が投稿される

**参加とキャンセル（PL向け）**
- Discord: 募集メッセージの「参加希望」ボタン
- Web: 募集詳細でログイン中の自分として応募／取消

目標人数に達すると「満員」になり、募集主へ Discord 通知。

---

**9. プレイヤー詳細**

- プロフィール編集、プレイスタイルレーダー、推しキャラ／最強シナリオ
- PL通過済（部内卓＋部活外の個人履歴）／GM経験済シナリオ
- GM可能シナリオの登録（「気になる」通知の宛先）
- 所持キャラクター一覧とスケジュール

※ 部活外シナリオはログインなしで「部活外を追加」から登録でき、ほかのプレイヤーの履歴も追加・削除可。シナリオ一覧には出ません。

---

**10. コメント・なりきりチャット**

- **コメント:** 各詳細画面や HOME の新着コメントで感想・やり取り
- **なりきりチャット:** 自分のキャラ名義で掲示板に投稿（他人のキャラでは投稿不可）

---

**11. よくあること**

- **ログインしたが個人画面が出ない:** 名簿未連携。ホーム上部バナーから自分の名前で連携
- **名簿に自分の名前が無い:** 管理者にプレイヤー追加を依頼
- **Discord ID がコピーできない／取れない:** Logout → Discord Login をやり直す
- **予定カレンダーが赤い:** その日の予定が未入力。日付を選んで昼／夜を入力
- **気になるを押しても DM が来ない:** すでに ON だった場合や、そのシナリオに GM可能登録者がいない場合は通知なし（新規 ON 時のみ）
